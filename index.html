HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • '!DOCTYPE html' is a delcaration. It is letting the browser know to expect HTML on the page
  • HTML elements are case sensitive
  • An element refers to the entire element from opening tag to closing tag
  • A tag refers only to what appears inside the angle brackets
  • A child element is nested inside other elements
  • A parent element holds all the children (ex. the 'body' element
  • Semantic HTML elements are the name of the elements that describe its content and the role it plays on a webpage (ex. 'body')

  • 'head'elements defines three elements:
      • charset sets our character encoding
      Http-equiv element sets an HTTP header for the content
      viewport element sets the default view for our webpage

  • A header must include:
      • !DOCTYPE html
      html
      meta
      title
