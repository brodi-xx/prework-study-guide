HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • '!DOCTYPE html' is a delcaration. It is letting the browser know to expect HTML on the page
  • HTML elements are case sensitive
  • An element refers to the entire element from opening tag to closing tag
  • A tag refers only to what appears inside the angle brackets
  • A child element is nested inside other elements
  • A parent element holds all the children (ex. the 'body' element
  • Semantic HTML elements are the name of the elements that describe its content and the role it plays on a webpage (ex. 'body')

  • 'head'elements defines three elements:
      • charset sets our character encoding
      Http-equiv element sets an HTTP header for the content
      viewport element sets the default view for our webpage

  • A header must include:
      • !DOCTYPE html
      html
      meta
      title

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • Inline CSS applies to a specific HTML tag. They are useful for quick and permanent changes but are less flexible than external and internal. Each inline style must be edited individually when changes are made.
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page. A single change to the CSS rule will apply to all tagged page elements.
  • External CSS style sheet is the most common way to style a webpage. It allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • Declarations contain two components: the CSS property and the value of the property.
  • The padding property adds space around the content inside an element. A margin is a transparent buffer outside an element.
  • * applies a rule to all elements visible on the page. In this rule, we have declared that ALL of the elements on the page will have the style properties listed in it.
